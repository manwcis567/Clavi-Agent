让 Agent 不只会回答，而是真的去完成任务

Clavi Agent 是一个极简但专业的 Agent 框架，围绕真实执行而不是演示话术构建。它把文件、Shell、Session Memory、Skills、MCP 和长上下文管理串成一个稳定的执行闭环，适合快速上手，也适合继续二次开发。
